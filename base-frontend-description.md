Base Frontend:

Built as a custom card (deployed via HACS, hosted on Github) or as a custom panel in Home Assitant

Users input their appliance preferences (high/low hourly temperatures, battery chagre requirements and timing, home location/electric price data)

Calls our hungrymachines-api backend to optimize electric appliance use

Built around our API Contract (we will offer this base version but would like other people to also be able to build custom front end designs)
